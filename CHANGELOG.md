## [1.0.0] - 2026-03-16

### Added
- Day-by-day itinerary builder with hotels, activities, budget and logistics
- Live budget breakdown (hotels, activities, general items, grand total)
- Calendar grid view for the whole trip at a glance
- PDF and CSV export
- Google Maps links auto-generated for hotels, activities and full route
- AI copilot via [Ollama](https://ollama.com) — full itinerary generation and conversational chat
- Import JSON itineraries from any external AI (ChatGPT, Claude, etc.)
- MCP server for agentic access from Claude Code or any MCP-compatible client
- 74-test suite with GitHub Actions CI (lint + tests on every push)
- Apache 2.0 licence
